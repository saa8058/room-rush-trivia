# Atwix Trivia Sounds

Place optional, royalty-free audio files here with these exact names:

- `countdown.mp3`
- `answer-lock.mp3`
- `reveal.mp3`
- `correct.mp3`
- `wrong.mp3`
- `leaderboard.mp3`
- `winner.mp3`
- `background-loop.mp3`

The app already works without these files. Missing sound effect files fall back to short generated tones so the audio controls can be tested immediately. Missing background music stays silent; the app never creates a repeating music fallback.

Keep background music quiet and loop-friendly. Avoid copyrighted music or sound effects.
